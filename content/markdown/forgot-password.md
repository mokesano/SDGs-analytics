# Reset Your Password

Forgot your password? No problem. Enter your email address and we'll send you instructions to reset it.

## Password Recovery

### Steps to Reset

1. **Enter Email** - Provide the email address associated with your account
2. **Check Inbox** - Look for an email from Wizdam AI (check spam folder too)
3. **Click Link** - Use the secure reset link in the email (valid for 1 hour)
4. **New Password** - Create a new strong password

### Security Notes

- Reset links expire after **60 minutes** for security
- Each link can only be used **once**
- If you don't receive an email, check your spam folder
- Contact support if you continue having issues
